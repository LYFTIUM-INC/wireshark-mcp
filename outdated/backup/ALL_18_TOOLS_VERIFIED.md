## 🚀 **EXAMPLE USAGE**

### Extract Network Activity from Specific Time Window
```python
await wireshark_pcap_time_slice(
    input_file="traffic.pcap",
    start_time="2025-01-08T14:30:00", 
    end_time="2025-01-08T15:30:00"
)
```

### Detect DNS Tunneling Activity
```python
await wireshark_dns_analyze(
    input_file="traffic.pcap",
    analysis_type="tunneling",
    entropy_threshold=3.5
)
```

### Perform Multi-Host Network Capture
```python
await wireshark_remote_capture(
    capture_mode="multi",
    hosts=[
        {"host": "web01.company.com", "username": "netadmin"},
        {"host": "db01.company.com", "username": "netadmin"}  
    ],
    synchronized=True,
    duration=300
)
```
